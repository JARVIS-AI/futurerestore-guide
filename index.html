Introduction

Futurerestore is a tool that allows iOS devices to upgrade, downgrade, or re-restore to an unsigned iOS firmware using SHSH blobs. This is a guide will go over how to use Futurerestore, and the different methods you can use to restore your device to an unsigned firmware.

This guide is based off of this one, except it has been made into a website with better formating, making it much more simple to read and easier to follow.

Know that if you decide to follow this guide, take everything you read into consideration and be careful! No one will be held responsible for any damage YOU cause to your device, whether it was caused by you using the wrong file, or the wrong command.

Background Info

Before you begin following this guide, keep in mind that:

  • This guide is targeted to people who (as of August 2019) want to use Futurerestore to upgrade, downgrade, or re-restore to iOS 12.2. This doesn't make this guide limited to those users only, as the process is the same for other devices/firmwares, but with slight modifications.
  • As of August 2019, iOS 12.4's SEP and Baseband are fully compatible when upgrading, downgrading, or re-restoring to iOS 12.2, and the "fortnight bug" does NOT occur. This means that you can safely use the latest SEP and Baseband, instead of manually specifying a compatible SEP and Baseband when restoring, without having to worry about your device not functioning after 2 weeks.
  • If you're a Windows user, make sure iTunes and its drivers are installed correctly. This is very critical to you, as you want to avoid having the Microsoft Store version of iTunes.
  • If you decide to use a backup after using Futurerestore, make sure that you have gone through the process of getting rid of your jailbreak before backing up your jailbroken device. Modern jailbreaks have an option named "Restore RootFS" that can help you with this.
  • This process does not work for 32-bit Windows users. Sorry!

Getting ready to Restore

In order to futurerestore to your desired firmware, there are a few things you will need, such as:

  • Futurerestore's latest released version
  • Your SHSH blob file from TSSSaver (If you're unsure as to which file you should download, head to the noapnonce folder, and download your blob from there)
  • The IPSW containing an SEP and Baseband that is compatible with the version you're going to upgrade, downgrade, or re-restore to. If you're looking for beta firmwares, you can find them here, otherwise, head over to ipsw.me to obtain officially released firmwares

If you're going to specify your SEP and Baseband, you'll be required to extract the IPSW that contains the SEP and Baseband that is compatible with the version you're restoring to and obtain:

  • A .im4w file containing the boardconfig identifier of your device (This is your SEP)
  • A .bbtw file containing the iOS version number of the IPSW (This is your baseband)
  • The BuildManifest.plist file from said IPSW

You can use extract.me to get these files without having to actually extract the entire IPSW.

Once you have all of the files, its a good practice to put them inside one folder, which is often named futurerestore.

Finding your Nonce Generator

Finding your nonce generator and setting it is a critical step when attempting to Futurerestore, as that generator is essesially one of the main components that will allow you to Futurerestore to your desired firmware.

In order to find your nonce generator:

  1. Open your downloaded blob from TSSSaver with any kind of text editor
  2. (On Windows) Press Ctrl + F, (On macOS) Press Command + F, then search for the word "generator

Next to, or underneath the generator, you will see that there's a string of (around) 16 characters.Please save this string inside a text file that should also be inside your futurerestore folder. You will use this string later.

Setting your Nonce Generator

Now that you have found your nonce generator, its time to set it on your device. There are many nonce setters out there, which support many different firmwares. Here's a list of the most commonly used, made by known developers:

  • Ph≈ìnixNonce for iOS 9.x.
  • NonceSet for iOS 10.x.
  • NonceSet1112 for iOS 11 - 11.1.2.
  • GeoNonceSetter12 for iOS 12 - 12.2.
  • System Info for iOS 8, 9, 10, and 11. (Available in Cydia, or any other package manager)

Alternatively, modern jailbreak tools, such as unc0ver, or Chimera, have an option to set a nonce generator on support firmwares, so you can use them to set your nonce as well.

Here's how you can set-up your nonce:

  1. Open your nonce setter tool (or modern jailbreak tool)
  2. Find a text field with the words "generator", "nonce" or "set nonce"
  3. Within that region, type in your key that you saved from your blob
  4. Press the "Set Nonce" button (or hit "Jailbreak" if you're using a jailbreak tool)

Once you've set your nonce generator, you're ready to begin the restoration process. Do note that if you're device powers off at this point for any reason, you will need to set up your nonce generator again.

Restoration

You're finally ready to restore your device using Futurerestore. Before restoring, there are things you should keep in mind that:

  • Failing to disable "Find my iPhone/iPad/iPod and/or having NO PLS Recovery installed before beginning the restoration process can cause the restore not to begin, or fail.
  • If your device is put into "Recovery Mode" during this process, do not panic. This is entirely normal. However, avoid any iTunes alerts by all means, as you don't want iTunes updating, or restoring your device.

With that out of the way, note that are many ways you can restore to an unsinged firmware. This guide will go through most (if not, all) ways of restoring.

Common Restore (Automatic SEP & Baseband)

This method is commonly used when the latest SEP and Baseband of the latest iOS version is compatible with the version you are trying to restore to.

In order to upgrade, downgrade, or re-restore using this method:

  1. Open Command Prompt/Terminal
  2. Open your futurerestore folder, then drag Futurerestore into Command Prompt/Terminal, then type -t
  3. Drag your blob file into Command Prompt/Terminal
  4. Type --latest-sep --latest-baseband, then drag your IPSW

In the end, your result should look something like this (assuming your folder is on your Desktop, using Windows):

C:\Users\User\Desktop\futurerestore\futurerestore.exe -t [your blob] --latest-sep --latest-baseband [IPSW]

For iPods and Wifi-Only iPads, you should not specify the latest baseband:

C:\Users\User\Desktop\futurerestore\futurerestore.exe -t [your blob] --latest-sep --no-baseband [IPSW]

Once you are sure that you have the right command, hit "Enter" and the restore should begin.

Manually specifying SEP & Baseband

This method is used when the latest SEP and Baseband of the latest iOS version is not compatible with the version you are trying to upgrade, downgrade, or re-restore to.

When specifying a "custom" SEP and Baseband, you are essensially doing so solemly because that specific SEP and Baseband is compatible with the version you're trying to upgrade, downgrade, or re-restore to.

In order to upgrade, downgrade, or re-restore using this method:

  1. Open Command Prompt/Terminal
  2. Open your futurerestore folder, drag Futurerestore into Command Prompt/Terminal, then type -t
  3. Drag your blob file into Command Prompt, then type -s, then drag your SEP file into Command Prompt/Terminal
  4. Type -b, then drag your baseband file intro Command Prompt/Terminal
  5. Type -p, then drag your BuildManifest file into Command Prompt/Terminal
  6. Type -m, drag your BuildManifest file (again), then drag your IPSW into Command Prompt/Terminal

In the end, your result should look something like this (assumming your folder is in your desktop, using Windows):

C:\Users\User\Desktop\futurerestore\futurerestore.exe -t [your blob] -s [SEP] -b [Baseband] -p [BuildmManifest] -m [BuildManifest] [IPSW]

For iPods and Wifi-Only iPads, you should not specify your baseband:

C:\Users\User\Desktop\futurerestore\futurerestore.exe -t [your blob] -s [SEP] --no-baseband -p [BuildmManifest] -m [BuildManifest] [IPSW]

Once you are sure that you have the right command, hit "Enter" and the restore should begin.

If you run into any issues during the restoration process, please refer to this page, which has the most common errors you can encouter when using Futurerestore and how to solve them.
